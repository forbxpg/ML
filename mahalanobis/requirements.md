Задания

1. Рассчитать расстояния Махаланобиса от каждой точки до каждого из классов при различных предположениях: а) ковариационные матрицы всех классов равные, скалярные; б) равные, диагональные; в) различные, скалярные; г) различные, диагональные; д) равные; е) различные. Визуализировать матрицы для каждого случая. 2. Построить диаграммы рассеяния расстояний (по оси абсцисс 1 – расстояние до положительного класса (с меткой +1), по оси ординат 2 – до отрицательного класса) для каждого случая. Нанести на диаграмму прямую 1 = 2. Построить гистограммы распределения расстояний данных каждого класса до положительного класса для случаев а)–е).
2. На диаграмме рассеяния исходных данных изобразить границы классов (геометрическое место точек, равноудалённых по Махаланобису от обоих классов). Сравнить границы для случаев а)–е).
3. Рассчитать показатели точности (error rate, accuracy, sensitivity, specificity, precision, fall-out, F1-score, Cohen’s kappa) на обучающей и тестовой выборках для построенных классификаторов (для случаев (а)–(е)).
4. Построить ROC-кривые и PR-кривые и рассчитать ROC AUC и PR AUC на обучающей и тестовой выборках для случаев а)–е).
5. Сделать выводы о влиянии предположений о модели обучающих данных на точность классификации по расстоянию Махаланобиса.

Указания
Кросс-валидация: Holdout (70/30).
В качестве классификационных очков при построении ROC-кривых и PR-кривых использовать разность между расстояниями Махаланобиса до положительного и отрицательного классов.
